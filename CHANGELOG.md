# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added

- Initial public release of Character Toolkit.
- **Character analysis:** `characterCount`, `letterCount`, `digitCount`, `specialCharacterCount`, `uppercaseCount`, `lowercaseCount`, `vowelCount`, `consonantCount`, `whitespaceCount`, `tabCount`, `lineCount`, `emojiCount`, `punctuationCount`, `uniqueCharacterCount`, `characterFrequency`.
- **Word analysis:** `wordCount`, `sentenceCount`, `paragraphCount`, `longestWord`, `shortestWord`, `averageWordLength`, `uniqueWords`, `duplicateWords`, `wordFrequency`, `mostCommonWords`.
- **Statistics:** `readingTime`, `speakingTime`, `textDensity`, `lexicalDiversity`, `textComplexity`, `entropyScore`, `keywordDensity`.
- **Rearrangement:** character and word reverse/shuffle/sort/rotate/move/swap helpers.
- **Palindrome & anagram** utilities.
- **Case conversion** (camel, pascal, snake, kebab, title, sentence, invert, toggle).
- **Text cleaning** helpers.
- **Extraction:** emails, URLs, domains, phone numbers, hashtags, mentions.
- **Encoding:** base64 and URL encode/decode.
- **Validation:** email, URL, numeric, alphabetic, alphanumeric predicates.
- **Similarity:** Levenshtein, Hamming, normalized similarity score.
- **Advanced:** n-gram generation and aggregate report builders.
- Full Vitest test suite, JSDoc documentation, and per-tool docs.
